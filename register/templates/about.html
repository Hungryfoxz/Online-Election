'About Us'

schoolJorhat Institute of Science and Technology

Jorhat Institute of Science and Technology, in short, JIST is a premier and unique Institute of Assam, situated in Sotai, Jorhat District of Assam. This web application is the property of JIST.

The website as you may have already guessed is an polling web application.It was developed by one of our students, which by the way came as a part of a competition.

The purpose of this website is to hold only general elections of the Institution, which are organised every year.

codeJISTSU Application

The website is build on the Django framework, which is developed from the Python language. So the backend of this website is nothing but Python. It uses SqlLite as the Database.It pulls the Django's Templating engine to render into the HTML document.

The frontend is HTML,Css,Vanilla JS & JQuery.It makes use of the Css framework MaterializeCSS.

Apart from this only some AJAX requests are made for the JSON data.

visibility_offSecurity

The website provides the maximum Security in terms of users PII information. All the user inputs are sanitized in order to protect itself from the possible XSS,Sql & Temlplate injection attacks.

It uses advanced sanitization features to detect unwanted characters being injected. CSRF token is also used to prevent unintended redirects.

If you think that you have found any severe bugs like RCE,Sensitive data exposure,Broken access control etc. Please report us immediately. We will be happy to resolve the issue as soon as possible. Also avoid unwanted testing, unless asked by the responsible authority.
